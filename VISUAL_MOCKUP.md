  Level 1 (Page):   #FAFAFA (Lightest grey)

## ğŸ­ Interactive States

### Button States
```
Default:    Background, no shadow
Hover:      Transform scale(1.05), soft shadow
Active:     Transform scale(0.98)
Disabled:   Opacity 0.5, no hover
```

### Table Row States
```
Default:    White/Slate background
Hover:      Light grey/Darker slate background
Selected:   Green tint (optional future feature)
```

### Filter Pills
```
Inactive:   Grey background, dark text
Active:     Green background, white text
Hover:      Slightly darker shade
```

  Small:   8px (rounded-lg) - buttons, inputs

  None:    No elevation
